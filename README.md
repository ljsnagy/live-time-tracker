# Live Time Tracker

Streamlined time tracking for bootcamp lessons.

livetimetracker.surge.sh

## Getting Started


### Installing

Install deps with NPM

```
npm i
```

Run the dev server

```
npm run dev
```

## Deployment

Run build

```
npm build
```

Deploy with your prefered static site host

## Built With

* [Webpack](https://webpack.js.org/)
* [Svelte](https://svelte.dev)
